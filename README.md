# UART Communication Protocol

## Overview
This project implements UART communication in Verilog HDL using Xilinx Vivado.

## Modules

- uart_top
- baudrate_gen
- transmitter_tx
- receiver_rx

## Simulation

A complete testbench is provided:

- uart_top_tb

## Tools Used

- Verilog HDL
- Xilinx Vivado 2018.2

## Features

- UART Transmission
- UART Receiver
- Baud Rate Generator
- Simulation Verified
