# UART Communication Protocol

UART Transmitter and Receiver implementation in Verilog HDL using Xilinx Vivado.

## Project Structure

rtl/
- baudrate_gen.v
- transmitter_tx.v
- receiver_rx.v
- uart_top.v

sim/
- uart_top_tb.v

## Features

- UART Transmission
- UART Reception
- Baud Rate Generator
- Simulation Verified
